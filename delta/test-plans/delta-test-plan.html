Unofficial Client — Delta Test Plan

Manual QA Portfolio Deliverable

1. Introduction

This test plan outlines the verification and validation strategy for Delta Airlines’ core workflows, including login, flight search, booking, cancellation, and UI responsiveness. It covers functional behavior, UI consistency, error handling, and overall system responsiveness.

2. Scope

In Scope:

  • Functional tests: login, search flights, booking, cancellation
  • UI tests: layout, buttons, links, responsiveness
  • Positive and negative scenarios for each feature

Out of Scope:

  • Backend payment processing beyond UI confirmation
  • Third-party integrations not visible in UI
  • Load, performance, or stress testing

3. Test Objectives

  • Verify that users can log in successfully with valid credentials
  • Ensure invalid credentials produce correct error messages
  • Validate flight search and booking workflows
  • Confirm booking cancellation process works as expected
  • Ensure layout, fields, buttons, and links are displayed and functional
  • Capture visual confirmation of UI through screenshots

4. Test Approach

  • Manual QA testing
  • Exploratory, functional, and UI validation methods
  • Regression tests executed as needed
  • Grouped by feature: Login, Search Flight, Booking, Cancellation, UI
  • Screenshots of each step, censored where necessary

5. Test Environment

  • Browsers: Chrome (latest), Safari
  • OS: macOS, Windows 10+
  • Network: Standard broadband

6. Risks & Dependencies

  • Delta website UI may change at any time
  • Network outages may affect accessibility
  • Rate-limiting, captchas, or maintenance may interfere with test execution
  • Delays in test data availability

7. Resources

  • Tester email: heatqa.test@gmail.com
  • Test accounts with valid credentials
  • Flight search and booking test data

8. Test Deliverables

  • Test Cases
  • Bug Reports
  • Screenshots

9. Exit Criteria

  • All planned test cases executed
  • All critical and high-severity defects documented
  • Test execution summary reviewed and approved
