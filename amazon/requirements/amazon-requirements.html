REQ-AMZ-01: User Authentication & Accessibility

WCAG 2.1 AA Compliance
1. Description
The system shall allow users with valid email/password combinations to establish a secure authenticated session. Upon success, a personalized greeting "Hello, [User]" must be displayed.
2. A11y Compliance
Interface must meet WCAG 2.1 AA standards:
  • Keyboard-only navigation (Tab/Enter) must be fully functional.
  • Screen readers must announce input labels correctly.
  • Minimum color contrast ratio of 4.5:1 for all text.

TC-AMZ-001, TC-AMZ-007, TC-AMZ-008
