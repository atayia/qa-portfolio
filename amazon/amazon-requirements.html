REQ-AMZ-01: User Authentication & Accessibility

Verified
Description
The system shall allow users with valid email/password combinations to establish a secure authenticated session. Upon success, a personalized greeting "Hello, [User]" must be displayed.
  • A11y (WCAG 2.1 AA): Keyboard-only navigation (Tab/Enter) must be fully functional.
  • A11y (WCAG 2.1 AA): Screen readers must announce input labels correctly.
